Detailed Description

The purpose and scope of this file is to implement SYNC CT HAL, Prototype of the Sync CT Interface HAL This module implements contact interface Api's for synchronous profile in flash memory.

Contact ip can be used either in synchronous mode or asynchronous mode. This module designed and can be extended to multi-process environment This module contains only the hardware abstraction Api's The user of this module can use these Api's for contact protocol(I2C) development

§ phhalSyncCt_I2CPowerup()

phStatus_t phhalSyncCt_I2CPowerup ( uint8_t  bPullupSetStatus,
uint8_t  bCardPresConnectorStatus,
uint8_t  bSlewRateConfig 
)

This function is used to power up I2C card which consists IP initialization and card activation.

Precondition
The I2C card must be present in the slot before calling this API. phhalSyncCt_CheckCardPres() can be used to check pre-condition.
Parameters
[in]bPullupSetStatusIf logic set to 1 Pull up else pull down config.
[in]bCardPresConnectorStatusIf Logic 1 Normally open else normally close config.
[in]bSlewRateConfigTo rise from 0 to its selected value (see ct_pcr_reg register), the time needed by VCC depends on the capacitance.
Return values
PH_ERR_SUCCESSAlways.

§ phhalSyncCt_I2CExtendedRequestData()

phStatus_t phhalSyncCt_I2CExtendedRequestData ( uint8_t  bI2CExtended,
uint8_t *  pbReceiveBuf,
uint8_t *  pbDataBuffer 
)

This function is used to request or receive data from I2C card which works in Extended mode and can get data from random address location specified in user data buffer.

Parameters
[in]bI2CExtendedIt should be 1 to use I2C in extended mode.
[out]pbReceiveBufPointer to Receive Buffer in which bytes will store from card.
[in]pbDataBufferPointer to User data buffer in which Slave address ,Word address information and number of bytes to read information should be stored in given manner :
  • Slave address = pbDataBuffer[offset + 0].
  • Word address msb = pbDataBuffer[offset + 1].
  • Word address lsb = pbDataBuffer[offset + 2].
  • Data count msb = pbDataBuffer[offset + 3].
  • Data count lsb = pbDataBuffer[offset + 4].
Return values
PH_ERR_SUCCESSIf All data is read successfully.
PH_ERR_FAILEDIf All data is not read successfully.
maskedPH_ERR_SYNC_CT_I2C_BUFFER_TOO_SMALL If Number of bytes to Read is more than 512.
