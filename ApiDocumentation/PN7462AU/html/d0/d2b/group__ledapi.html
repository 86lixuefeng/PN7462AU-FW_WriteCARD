§ phLED_SetPattern()

void phLED_SetPattern ( const uint8_t  newPattern[PH_LED_BLINK_LEN])

Set the sequence of the LED Patterns to be shown.

Parameters
newPatternA array of #PH_LED_BLINK_LEN bytes. Each byte in the pattern shows which LED has to be on.

Some of the refernce patterns are

  • Yellow LED Blinking :
    {L___Y_,L_____,L___Y_,L_____,L___Y_, L_____ };
  • Yellow LED Always ON :
    {L___Y_,L___Y_,L___Y_,L___Y_,L___Y_, L___Y_ };
  • Yellow LED ON, with RED Blinking :
    {L___YR,L___Y_,L___YR,L___Y_,L___YR, L___Y_ };
