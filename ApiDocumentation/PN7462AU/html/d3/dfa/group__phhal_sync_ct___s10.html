Detailed Description

The purpose and scope of this file is to implement S10 synchronous card api.

This module implements S10 synchronous card Api's in flash memory. This module designed and can be extended to multi-process environment This module contains only the hardware abstraction Api's The user of this module can use these Api's for S10 Synchronous card.

phStatus_t phhalSyncCt_S10Powerup ( uint8_t  bPullupSetStatus,
uint8_t  bCardPresConnectorStatus,
uint8_t  bSlewRateConfig,
uint8_t *  pbAtrBuffer 
)

This function is used to power up S9 card which consists IP initialization and card activation.

Parameters
[in]bPullupSetStatusIf logic set to 1 Pull up else pull down config.
[in]bCardPresConnectorStatusIf Logic 1 Normally open else normally close config.
[in]bSlewRateConfigTo rise from 0 to its selected value (see ct_pcr_reg register), the time needed by VCC depends on the capacitance.
[out]pbAtrBufferPointer to the ATR buffer where the ATR header bytes will be stored
Return values
PH_ERR_SUCCESSActivation for S10 card is successful.
PH_ERR_FAILEDActivation for S10 card is not successful.
maskedPH_ERR_SYNC_CT_CARD_ABSENT If card is absent in main slot.

[in]dwByteCountNumber of byte count to be write in card other than control data address byte.
Return values

PH_ERR_FAILEDCommand processing is not done successfully.
