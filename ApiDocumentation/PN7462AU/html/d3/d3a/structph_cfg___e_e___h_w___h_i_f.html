Detailed Description

Structure: HIF Host interface configurations.

These values are primarily used for phExHif.

Field Documentation

§ bInterface

uint8_t phCfg_EE_HW_HIF::bInterface

Hif interfaces like I2C, SPI, HSU, USB or disabled.

  • 0 => Disabled
  • 1 => I2C
  • 2 => SPI
  • 3 => HSU
  • 4 => USB

§ bBufferType

uint8_t phCfg_EE_HW_HIF::bBufferType

Hif interfaces like I2C, SPI, HSU, USB or disabled.

  • 0 => E_BUFFER_FORMAT_FREE, transparent (generic HW protocol format).
  • 1 => E_BUFFER_FORMAT_FIXED, Header + Payload + Crc.
  • 2 => E_BUFFER_FORMAT_NATIVE, transparent with Size at 1st Word.

§ bStoreErrData

uint8_t phCfg_EE_HW_HIF::bStoreErrData

Store error data or discard error data.

  • 0 => Disabled
  • 1 => Enabled

§ bEnableVBUSPullDown

uint8_t phCfg_EE_HW_HIF::bEnableVBUSPullDown

If the user board takes more time than expected to detect discharge during self power mode, user can set this bit to discharge faster.

There is a possibility of performance improvement of detecting USB disconnected based on PULLDOWN enabled/disabled.
