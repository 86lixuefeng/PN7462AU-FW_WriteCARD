Detailed Description

ROM Interface (System Services) APIs.

The PN7462AU System Services are protected services that provide customers the following APIs

  1. CLIF ROM Intreface
  2. Miscellaneous Intreface
  3. PVDD Settings Intreface
  4. SECROW Intreface
  5. CLIF Test BUS Intreface
  6. USB Primary Downlaod Intreface

These APIs are implemented in the ROM Code with privileged r/w access rights to SECROW, DFT block, protected registers of CLIF and NXP EE Section.

Where applicable, the parameters passed should always be from RAM.
