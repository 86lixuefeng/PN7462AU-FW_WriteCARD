Detailed Description

System Services CLIF Card Mode Activate Sub-Module Control Data Structure.

This is used to configure the Card Mode Activate functionality and also obtain the internal states of the Activation sub-module. This is used as parameter in phhalSysSer_CLIF_CardModeActivate function.

Note
SC1053 - Can't Initialize the TX/RX buffer if Tranceive is Enabled
