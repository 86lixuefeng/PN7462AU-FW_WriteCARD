§ phhalSysSer_SetCPU_Reset()

phStatus_t phhalSysSer_SetCPU_Reset ( void  )

phhalSysSer_SetCPU_Reset performs cpu reset only and directly boots from Flash.

It is not expected to return if it return then it is an error scenario.

Returns
Status code of type phStatus_t.

phStatus_t phhalSysSer_SetFlashProgram ( uint32_t  dwFlashPageNumber,

phhalSysSer_SetFlashProgram performs a page write in Flash.

Parameters
dwFlashPageNumberPage flash number.
pdataPointer to the data in RAM.
Returns
Status of the API
Return values
PH_ERR_SUCCESSSuccess status if operation completed successfully else failure.
Warning
This API is not expected to be used directly by the programmer/developer. Programmer/developer is expected to use the APIs provided in FLASH HAL
