Detailed Description

Structure: PcrPwrDown See phhalPcr_PwrDown_Setting_t.

It is a 32 bit value bit-file created by ORing enums of type phhalPcr_PwrDown_Setting_t used to select which settings must be applied to reduce power consumption during Suspend
