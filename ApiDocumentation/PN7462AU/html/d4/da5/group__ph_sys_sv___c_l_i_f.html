Detailed Description

APIs and Structrues for CLIF related ROM Interface.

Prototype of the CLIF System Services API.

This module implements CLIF interface in ROM. This module is not designed for multi user/process environment.

Macro Definition Documentation

§ PH_SYSSV_CLIF_MAX_MIFARE_UID_SIZE

#define PH_SYSSV_CLIF_MAX_MIFARE_UID_SIZE   7

CLIF Maximum value of Mifare Classic UID Size supported This is a limit for UID size in phhalSysSer_CLIF_MF_Authenticate_Parameters_t.ab_Uid.

Currently only 7 byte UID is the maximum supported for Mifare Classic cards.

Enumeration Type Documentation

§ phSysSv_Clif_MF_Authenticate_Step_t

enum phSysSv_Clif_MF_Authenticate_Step_t

CLIF Mifare Authentication Steps This is used to control the Mifare Authentication sub-module in phhalSysSer_CLIF_MF_Authenticate function.

It defines the various stages in the Mifare Crypto Authentication process.

Enumerator
PH_SYSSV_CLIF_MFAUTH_DISABLE 

0x00

Disable the Mifare Authentication sub-module completely

PH_SYSSV_CLIF_MFAUTH_LOADKEY 

0x01

Initialize the Mifare Authentication sub-module.

Note
This step needs to be used before any Mifare Authentication command communication is started
PH_SYSSV_CLIF_MFAUTH_PREPARE 

0x02

Configure the Mifare Authentication sub-module to begin Mifare Crypto Encryption from the subsequent communication and to provide the RA , RB' and RB'' values used for subsequent steps in Mifare Authentication Process

PH_SYSSV_CLIF_MFAUTH_MAX 

0x03

Comparison Maximum for this Enum type

PH_SYSSV_CLIF_MFAUTH_U32 

Ensure 32 bit width of this register.

§ phhalSysSer_CLIF_MF_Authenticate()

phSysSv_Clif_MF_Authenticate_Parameters_t *  pData 
)

This function will enable the MiFare Crypto sub module which is used for encrypted communication.

Warning
Correct Sequence of the calls for this function with dwStep for each authentication cycle needs to be performed
  • PH_SYSSV_CLIF_MFAUTH_LOADKEY
  • PH_SYSSV_CLIF_MFAUTH_PREPARE
Note
This function depends upon Rng for random number generation used in Mifare Authentication.

§ phhalSysSer_CLIF_Config_TxRxBuffer()

phStatus_t phhalSysSer_CLIF_Config_TxRxBuffer ( phSysSv_Clif_TxRxBuffer_Parameters_t *  pData)

This function will set-up the Transmit and Receive Buffer.

Warning
Card Mode Activate Module must not be active else the TX buffer initialization will fail.
The phSysSv_Clif_TxRxBuffer_Parameters_t.pRxBuffer Field for RX buffer needs to be DWORD aligned
DO NOT set up the Tx buffer while CLIF is in TRANSMITTING state, and DO NOT set up the Rx Buffer while the Receiver is enabled.
