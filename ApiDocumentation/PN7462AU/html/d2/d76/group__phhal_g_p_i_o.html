§ phhalPcr_ConfigInput()

phStatus_t phhalPcr_ConfigInput ( uint32_t  dwGpioNumber,
uint32_t  bEnable,
uint32_t  bWakeup,
uint32_t  bEnableInterrupt,
uint8_t  bActiveLow,
uint8_t  bLevelSensitive,
uint8_t  bBothEdge 
)

This Api configures the GPIO as input.

Along with enabling/disabling the Gpio as input it can also be configured as a wake up source and can be enabled as interrupt.

Note
By default the Gpio number from 01 to 08 is configured as input and also the wake up is enabled.
Parameters
[in]dwGpioNumberGpio number which ranges form 1 to 12.
[in]bEnableIf this boolean variable is true then the provided Gpio pin(Gpio number) will be configured as input.
[in]bWakeupIf this boolean variable is true then the provided Gpio pin(Gpio number) will be configured as wake up source.
[in]bEnableInterruptIf this boolean variable is true then the provided Gpio pin(Gpio number) will be configured as interrupt source.
[in]bActiveLowIf this variable is true then the interrupt will be generated on Active_low/falling edge else Active_high/Rising edge.
[in]bLevelSensitiveIf this variable is true then the interrupt will be generated on Level sensitive else Edge sensitive.
[in]bBothEdgeIf this variable is true then the interrupt will be generated on both positive edge and negative else Not both edge triggered.
Returns
Return value
Return values
maskedPH_ERR_INVALID_PARAMETER Parameter has not been set correctly
PH_ERR_SUCCESSConfiguring the Gpio as input enable or disable successful

phStatus_t phhalPcr_ConfigPuPd ( uint32_t  dwGpioNumber,
uint32_t  blPullUpEn,
uint32_t  blPullDownEn 
)

This Api configures the internal pull-up and pull-down resistors.

The internal pull up and pull down can be enabled for each Gpio.

Parameters
[in]dwGpioNumberGpio number which ranges form 1 to 12.
[in]blPullUpEnIf this boolean variable is true then pull up will be enabled for that perticular Gpio. Else it will be disabled.
[in]blPullDownEnIf this boolean variable is true then pull down will be enabled for that perticular Gpio. Else it will be disabled.
Returns
Status of operation
Return values
maskedPH_ERR_INVALID_PARAMETER Parameter has not been set correctly
PH_ERR_SUCCESSConfiguring the pull-up and pull-down is successfull
