Detailed Description

The PAGEFLASH Read/Write APIs.

Note
: Physically PN7462AU has two different instances of Flash. Flash 0,

FLASH HAL exposes the PAGEFLASH as a single Flash memory, not as separate Flash 0, Flash 1 memory banks. It is possible to have only one write or read of PAGEFLASH at any moment of time.

§ phhalFlash_ReadBuffer()

phStatus_t phhalFlash_ReadBuffer ( uint8_t *  pbBuffer,
uint8_t *  pbFlash,
uint32_t  dwBytesToRead 
)

This function reads PAGEFLASH memory data.

Page Flash memory can be read like RAM, using pointers. The value that this API adds is that it takes care of avoiding memory access violations, even when one or both of the addresses are mis-aligned. Additionally, parameter checking can be enabled.

Parameters
[out]pbBufferPointer to buffer where read data will be copied.
[in]pbFlashValid address of a byte in the PAGEFLASH.
[in]dwBytesToReadNumber of bytes to be read.
Returns
Status of the Operation.
Return values
maskedPH_ERR_INVALID_PARAMETER if parameters passed are invalid.
PH_ERR_SUCCESSif PAGEFLASH Read Operation completed successfully.

§ phhalFlash_WritePage()

phStatus_t phhalFlash_WritePage ( uint32_t  dwPageNumber,
uint8_t *  pbBuffer 
)

This function writes a single full 128byte PAGEFLASH page.

Note
FLASH pages are numbered from 0 onwards, but in User mode, Page flash pages that can be used range from PHHAL_FLASH_USER_PAGE_START to PHHAL_FLASH_USER_PAGE_END
The Write is synchronous i.e.Polling based. The API will return only after the write operation completes.
This API internally uses a System Service to write each Page in the Flash.
Parameters
[in]dwPageNumberValid index of a 128byte page in the PAGEFLASH.
[in]pbBufferPointer to buffer containing data to be written to PAGEFLASH. Must be 4byte aligned address.
Returns
Status of the Operation.
Return values
PH_ERR_SUCCESSif PAGEFLASH Write Operation completed successfully.
maskedPH_ERR_INVALID_PARAMETER if parameters passed are invalid and parameter checking is enabled in build.
SystemService error code in case call to System service fails.

§ phhalFlash_WriteBuffer()

phStatus_t phhalFlash_WriteBuffer ( uint8_t *  pbBuffer,
uint8_t *  pbFlash,
uint32_t  dwBytesToWrite 
)

This function writes data to PAGEFLASH.

This API can be used to write data of any valid size(even mis-aligned) to any valid address in the PageFlash(even mis-aligned addresses).

Note
The Write is synchronous i.e. The API will return only after the write operation completes.
This API internally uses a System Service to write each Page in the Flash.
Parameters
[in]pbBufferpointer to buffer containing data to be written to PAGEFLASH. Must be 4byte aligned address.
[in]pbFlashValid address of a byte in the PAGEFLASH. Must be 4byte aligned address.
[in]dwBytesToWritenumber of bytes to be written. Must be Non zero, multiple of 4.
Returns
Status of the Operation.
Return values
PH_ERR_SUCCESSif PAGEFLASH Write Operation completed successfully.
maskedPH_ERR_INVALID_PARAMETER if parameters passed are invalid.
SystemService error code in case call to System service fails.
