uint32_t phFlashBoot_Event_SysMsg_t::dwMsg[2]

Application data field.

SysTask may have messages flowing from all modules with different data streams (like structures, enums etc.). It is better to allocate a generic memory array where all the data streams can fit. To start with, assigned 8 bytes, word aligned.
