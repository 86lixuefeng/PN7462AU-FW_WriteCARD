§ phhalPcr_EnterLowPowerMode()

phStatus_t phhalPcr_EnterLowPowerMode ( phhalPcr_RfLdtParams_t *  psRfLdtParams,
phhalPcr_WakeUpConfig_t *  psWakeUpConfig,
phhalPcr_StandbyPrevReason_t *  psPreventionReason 
)

Entering into the low power mode Function This function is used to enter into the low power mode.

The low power mode can be either standby or suspend. The needed wake up sources can be enabled by the wake up config structure.

Warning
At least one wake up source has to be selected before calling this function.
If any of the HIF interfaces selected as wake up source user must make sure that the same HIF is selected before.
Parameters
[in]psRfLdtParamsParameters to be set during the RfLd wake up
[in]psWakeUpConfigPointer to the structure of the wake up selection sources
[out]psPreventionReasonPointer to the standby/Suspend prevention reason in case entering into the low power mode fails
Returns
Return Value
Warning
This function will not return if it enters successfully into the standby
Return values
PH_ERR_SUCCESSEntering the low power mode success
maskedPH_ERR_INVALID_PARAMETER Parameter has not been set correctly
maskedPH_ERR_PCR_NO_HIF_SELECTED No HIF is selected, but wake up source selected as one of the HIF
maskedPH_ERR_PCR_WUC_PARAM_ERROR Wake up source selected is wake up timer but the wake up timer value is zero
maskedPH_ERR_PCR_NO_TVDD_MON_ENABLED Wake up source selected is TVDD MON, but TVDD monitoring is not enabled
maskedPH_ERR_PCR_NOWAKEUP_ENABLED No wake up source is selected
