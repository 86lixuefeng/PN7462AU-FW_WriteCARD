Detailed Description

The purpose and scope of this file is to implement S9 Synchronous card Apis.

This module implements S9 synchronous card Api's in flash memory. This module designed and can be extended to multi-process environment The user of this module can use these Api's for contact protocol(T=0 or T=1) development

phStatus_t phhalSyncCt_S9Powerup ( uint8_t  bPullupSetStatus,
uint8_t  bCardPresConnectorStatus,
uint8_t  bSlewRateConfig,
uint8_t *  pbAtrBuffer 
)

This function is used to power up S9 card which consists IP initialization and card activation.

Parameters
[in]bPullupSetStatusIf logic set to 1 Pull up else pull down config.
[in]bCardPresConnectorStatusIf Logic 1 Normally open else normally close config.
[in]bSlewRateConfigTo rise from 0 to its selected value (see ct_pcr_reg register), the time
[out]pbAtrBufferPointer to the ATR buffer where the ATR header bytes will be stored
Return values
PH_ERR_SUCCESSActivation for S9 card is successful. PH_ERR_FAILED Activation for S9 card is not successful.
masked# PH_ERR_SYNC_CT_CARD_ABSENT If card is absent in main slot.

§ phhalSyncCt_S9_Write()

phStatus_t phhalSyncCt_S9_Write ( uint8_t  bWriteType,
uint32_t  dwOffset,
uint32_t  dwCardAddress,
uint32_t  dwByteCount,
uint8_t *  pbDataBuffer 
)

This function is used to write data from user buffer to card address.

Parameters
[in]bWriteTypeIt indicates for what mode we want to write in card.
  • 0x31 For Write in 9 bit mode (protected mode)
  • 0x33 For Write in 8 bit mode (unprotected mode)
[in]dwOffsetoffset in user buffer from where bytes are needed to start writing in card.
[in]dwCardAddresscard Address where bits are needed to write.
[in]dwByteCountNumber of byte count to write.
[in]pbDataBufferPointer to the user data buffer.
Return values
PH_ERR_SUCCESSIf given number of bytes have been write successfully.
maskedPH_ERR_SYNC_CT_WRITING_WITHOUT_PIN_VERIFIED If writing is attempted without pin is verified.
maskedPH_ERR_SYNC_CT_PROTECTED_BYTE If writing at protected byte place.
maskedPH_ERR_SYNC_CT_WRITE_ERROR If given number of bytes have not been write successfully.
