Detailed Description

LED Handling.

phLED.h: Generic helper utility to generated specific pattern on LEDs connected to bard.

Warning
Specific GPIOs which are connected to specific LEDs are switched on and off using this utility. Hence the behaviour is tightly coupled with the board configuration.

This module expects that

Project: PN7462AU

To use this module,

  1. Ensure that the module is already initialized with phLED_Init
  2. Identify a pattern of #PH_LED_BLINK_LEN bytes e.g.
/ * keep Green On and blink, yellow and red LED * /
static const uint8_t gkphLED_SoFarSoGood[PH_LED_BLINK_LEN] = {L__GY_,L__G_R,L__GY_,L__G_R,L__GY_,L__G_R};
  1. Initialize the pattern with phLED_SetPattern before the operation.
phLED_SetPattern(gkphLED_SoFarSoGood);
  1. Invoke phLED_TimerCallback either from a running timer or at every step of the ongoing operation.
phLED_SetPattern()
