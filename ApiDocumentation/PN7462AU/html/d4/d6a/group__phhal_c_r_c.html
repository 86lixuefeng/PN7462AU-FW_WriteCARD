Detailed Description

The Cyclic Redundancy Check APIs.

This module implements CRC interface Flash code HAL, specific to CRC module.

Warning
If the build Macro PHFL_PARAM_CHECK is less than PH_PARAM_CHECK_VERBOSE, then the input parameters will not be checked for erroneous values in the HAL API. In such a build configuration use the API with caution and pass valid parameters only. Invalid parameters will likely result in hardfaults or other failures.

This API calculates the CRC16/32 with LSB/MSB First for a arbitrary length of buffer.

Parameters
[in]dwCrcPreloadInitial Preset Value.
[in]bCRCModeCRC16/32 and MSB/LSB First.
[in]pbBufferPointer to buffer over which CRC is calculated.
[in]dwDataLengthLength of the buffer on which CRC is calculated
[out]pdwCrcValuePointer to a uint32 variable that will hold the CRC16/32 result
Returns
Status of the Operation
Return values
PH_ERR_SUCCESSIf CRC is calculated successfully .
maskedPH_ERR_INVALID_PARAMETER - Invalid passing parameter error.

§ phhalCrc_CheckCrc

#define phhalCrc_CheckCrc (   dwCrcPreload,
  bCRCMode,
  pbBuffer,
  dwDataLength 
)

This API checks the CRC16/32 with LSB/MSB First for a arbitrary length of buffer with provided CRC The provided CRC is padded to the end of the buffer.

Parameters
[in]dwCrcPreloadInitial Preset Value.
[in]bCRCModeCRC16/32 and MSB/LSB First.
[in]pbBufferPointer to buffer containing contents as well as CRC of the content
[in]dwDataLengthLength of the buffer on which CRC is checked
Returns
Status of the Operation
Return values
PH_ERR_SUCCESSIf CRC is Check is success .
maskedPH_ERR_INVALID_PARAMETER - Invalid passing parameter error.
maskedPH_ERR_FAILED - If CRC Check is failed.
